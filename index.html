shumaispeechsynthesis

v0.1.0 (Stable)

Python向けの超軽量・独立型音声合成エンジン

概要

shumaispeechsynthesis は、特定の環境に依存せず、ネットワーク接続も不要な音声合成ライブラリです。
あらかじめ録音された高品質な音声データを連結する方式を採用しており、低スペックなハードウェアでもリアルタイムに近いレスポンスを実現します。

特徴

📁 超軽量設計
ライブラリ全体で約500KB。ディスク容量を圧迫せず、配布も容易です。
🔌 完全オフライン
外部APIを使用しないため、インターネット環境がない場所（オフライン環境）でも動作します。
🤖 Raspberry Pi 最適化
CPU負荷が非常に低く、シングルボードコンピュータでのAIスピーカー開発等に最適です。

インストール

pipを使用してGitHubリポジトリから直接インストールできます。

pip install git+https://github.com/shumaistudio/shumaispeechsynthesis.git

クイックスタート

import shumaispeechsynthesis as ss

# 漢字・ひらがな混じりのテキストを読み上げます
ss.speechsynthesis("こんにちは。今日はいい天気ですね。")
